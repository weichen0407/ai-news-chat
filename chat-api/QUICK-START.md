# 🚀 Story Creator 快速入门

欢迎使用Story Creator！这是一个完全独立的剧情和NPC创作工具。

---

## ⚡ 5分钟上手

### 1. 启动应用 ✅

应用已经在运行！打开浏览器访问：

```
http://localhost:5001
```

### 2. 创建你的第一个剧情 📖

点击左侧的"➕ 新建剧情"按钮：

```
剧情名称：办公室风云
描述：互联网公司的职场故事
事件背景：某科技公司产品上线前夕，产品经理和程序员为功能实现发生分歧...
对话密度：3
图标：💼
```

点击"创建"！

### 3. 添加NPC角色 🤖

在剧情详情页面，点击"➕ 添加NPC"：

**角色1：产品经理**
```
名称：张伟
头像：👨‍💼
人设：产品经理，30岁，追求完美用户体验，经常在最后时刻改需求
```

**角色2：程序员**
```
名称：李娜
头像：👩‍💻
人设：资深程序员，28岁，技术能力强，不喜欢频繁改需求
```

### 4. 添加Prompt模板 💬

点击"➕ 添加Prompt"：

```
类型：系统提示
内容：请模拟职场对话，语言要贴近现实，体现角色的性格特点。
说明：基础系统提示
```

### 5. 导出剧情 📤

点击右上角"📤 导出"按钮，下载JSON文件。

## 💡 使用技巧

### 1. 剧情背景要详细
好的背景描述会让AI生成更真实的对话：

❌ 不好：两个人吵架
✅ 好：产品上线前一天，产品经理临时要求修改核心功能，程序员认为时间不够会影响质量

### 2. NPC人设要有特点
让每个角色有鲜明的性格：

❌ 不好：一个普通人
✅ 好：35岁技术总监，曾是创业失败者，现在保守谨慎，不愿冒险

### 3. 使用多种Prompt类型
- **系统提示**：控制整体风格
- **角色设定**：加强角色特征
- **对话引导**：引导剧情发展
- **情节推进**：触发转折点

## 🔧 常用操作

### 快速创建剧情流程
1. 点击"新建剧情" → 填写基本信息 → 创建
2. 点击"添加NPC" → 连续添加多个角色
3. 点击"添加Prompt" → 设置对话模板
4. 点击"导出" → 保存JSON文件

### 编辑和删除
- ✏️ 点击编辑按钮修改内容
- 🗑️ 点击删除按钮移除项目

### 查看统计
顶部显示：
- 📖 总剧情数
- 🤖 总NPC数
- 💬 总Prompt数

## 📦 导出的文件用途

3. **AI集成** - 导入到AI对话系统
4. **版本管理** - 保存不同版本

## 🐛 遇到问题？

### 应用没反应
- 刷新页面 (Ctrl+R / Cmd+R)
- 检查浏览器控制台

### 数据丢失
- 数据保存在 `db/stories.db`
- 定期备份这个文件

### 端口被占用
- 修改 `server.js` 中的 `PORT` 值
