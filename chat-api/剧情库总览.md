## 🎯 剧情特点

### ✅ 基于真实事件
- 绝命毒师（经典美剧）
- 甄嬛传（热门国产剧）
- NBA总决赛（真实体育赛事）
- FTX崩盘（2022年真实事件）
- 美国大选（2024年进行中）
- 马斯克与特朗普（真实合作）
- 中美科技战（当前时事）
- OpenAI宫斗（2023年真实事件）
- 瓦格纳叛乱（2023年真实事件）
- 明星塌房（娱乐圈常见现象）

### ✅ 角色立体丰富
每个角色都有完整的10项属性：
- 基本信息（姓名、年龄、职业）
- 性格特点
- 技能专长
- 生活习惯
- 喜好与厌恶
- 背景故事
- 目标追求
- 内心恐惧

### ✅ 冲突激烈抓马
- 权力争夺（办公室、宫廷、政治）
- 道德困境（毒品、欺骗、背叛）
- 竞技对抗（体育、商业、战争）
- 理想冲突（AI安全、民主、技术）
- 人性考验（贪婪、恐惧、欲望）

### ✅ 开放式结局
每个剧情都可以走向多个方向：
- 正义胜利 vs 权谋得逞
- 和解妥协 vs 鱼死网破
- 黑化复仇 vs 救赎成长
- 意外反转 vs 按部就班

## 🚀 快速开始

### 访问应用
```bash
http://localhost:5001
```

### 重新创建剧情
```bash
# 创建所有11个剧情
npm run seed

# 或者分别创建
npm run seed:all      # 创建10个热门剧情
npm run seed:office   # 只创建办公室剧情
```

### 数据库迁移
```bash
npm run migrate  # 添加新字段
```

你也可以基于这些剧情创建自己的版本：

1. 访问 http://localhost:5001
2. 点击"创建新剧情"
3. 选择一个模板或从零开始
4. 修改角色属性、添加新角色
5. 设置剧情背景和冲突
6. 保存并分享
